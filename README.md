# CS1 Learning Through Game Creation

This project is an interactive, browser-based learning resource designed to teach introductory programming concepts through guided game creation.

The project includes a playable game and instructional modules covering variables, conditionals, loops, collision detection, functions, and debugging.

## Live Demo
https://savitamanchanda.github.io/cs6460-game-learning/

## Project Structure

- `index.html` – Main interactive game
- `game.js` – Game logic and behavior
- `style.css` – Styling
- `modules/` – Instructional modules (Module 1–6)

## Learning Objectives

This project helps learners understand:

- Program state (variables)
- Input handling and conditionals
- Game loop and iteration
- Collision detection and scoring
- Code organization using functions
- Debugging and reasoning about program behavior

## How to Use

1. Open the live demo link
2. Use arrow keys to move the player
3. Explore each module to understand how the game works
4. Modify code and observe how behavior changes
