What is the game loop?

The game loop is the part of the program that keeps running over and over. It updates the game state, redraws the screen, and creates the sense of motion.

In this project, the loop makes the orange orb move automatically and updates the frame counter continuously.

Code to study

function gameLoop() {
  frameCounter++;

  clearCanvas();
  drawGrid();
  drawInstructions();

  updatePlayer();
  updateOrb();

  drawPlayer();
  drawOrb();

  updateInfoPanel();

  requestAnimationFrame(gameLoop);
}

Why iteration matters

The loop repeats these same steps many times each second. That repetition is what makes the orb move smoothly across the screen instead of only changing once.
