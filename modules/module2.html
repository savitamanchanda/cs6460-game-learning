How does the player move?

The program checks whether a key is being pressed and uses if statements to decide how the player position should change.

Movement logic

if (keys.ArrowUp) {
  playerY -= playerSpeed;
}

if (keys.ArrowDown) {
  playerY += playerSpeed;
}

if (keys.ArrowLeft) {
  playerX -= playerSpeed;
}

if (keys.ArrowRight) {
  playerX += playerSpeed;
}

Boundary logic

These conditionals stop the player from moving outside the canvas. 

if (playerX < 0) {
  playerX = 0;
}

if (playerY < 0) {
  playerY = 0;
}

if (playerX + playerSize > canvas.width) {
  playerX = canvas.width - playerSize;
}

if (playerY + playerSize > canvas.height) {
  playerY = canvas.height - playerSize;
}
