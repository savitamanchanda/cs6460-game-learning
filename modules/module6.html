What is debugging?

Debugging means identifying why a program behaves differently from what you expected and then tracing that behavior back to the code.

Debugging activity 1

Imagine the orb stops moving after the game loads. Look at this code: 

function gameLoop() {
  frameCounter++;

  clearCanvas();
  drawGrid();
  drawInstructions();

  updatePlayer();
  // updateOrb();

  drawPlayer();
  drawOrb();

  updateInfoPanel();

  requestAnimationFrame(gameLoop);
}

What is causing the issue?
