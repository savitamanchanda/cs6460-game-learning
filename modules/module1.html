What is program state?

Program state means the values a program is currently storing. In this game, variables such as playerX, playerY, playerSize, playerSpeed, and playerColor control what the learner sees on screen.

Code to modify

Open game.js and find this section:

// MODULE 1: change values here to experiment
let playerX = 80;
let playerY = 120;
let playerSize = 40;
let playerSpeed = 5;
let playerColor = "royalblue";

Try these changes

  • Change playerSpeed from 5 to 10
  • Change playerSize from 40 to 60
  • Change playerColor from "royalblue" to "tomato"
  • Change the starting position values for playerX and playerY
