Playable Game Prototype

Use the arrow keys to move the blue square around the screen.

Try to guide the square toward the orange orb. When the square touches the orb, your score will increase and the orb will move to a new position.

As you go through the modules, you’ll explore how this game works by changing values, observing how the game updates each frame, understanding how collisions are detected, and gradually organizing the code into functions.

Pay attention to how your actions affect what happens on the screen which will help you connect the code to the behavior you see.

Watch how the score and collision status change as you play.

How to interact with this project

  • Start by playing the game: Use the arrow keys to move the blue square and collect the orange orb.
  • Module 1: Change player values such as speed, size, color, or starting position.
  • Module 2: Look at the if statements that control movement and boundaries, then test what changes when one is removed.
  • Module 3: Change the orb speed and watch how the loop continuously updates the game.
  • Module 4: Observe how touching the orb changes the score and causes the orb to reset.
  • Module 5: Identify which functions control drawing, movement, collision, and score updates.
  • Module 6: Work through the debugging prompts and explain which part of the code you would check first when behavior is wrong.
