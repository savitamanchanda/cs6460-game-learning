Playable Game Prototype

Use the arrow keys to move the blue square around the screen.

Try to guide the square toward the orange orb. When the square touches the orb, your score will increase and the orb will move to a new position.

As you go through the modules, you’ll explore how this game works by changing values, observing how the game updates each frame, understanding how collisions are detected, and gradually organizing the code into functions.

Pay attention to how your actions affect what happens on the screen which will help you connect the code to the behavior you see.

Watch how the score and collision status change as you play.
